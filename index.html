Centos7虚拟机下groovy安装

安装命令
curl -s get.sdkman.io | bash
source “/root/.sdkman/bin/sdkman-init.sh”
sdk install groovy

检查版本

设置环境变量
export GROOVY_HOME=/usr/local/groovy-3.0.9
export PATH=${GROOVY_HOME}/bin:${PATH}
